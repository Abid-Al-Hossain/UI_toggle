# UI Foundry Toggle / Switch Component Generator

Standalone visual toggle generator built with Next.js.

This app is a focused switch-building studio for binary on/off controls, branded toggles, and stylized switch surfaces that can be edited visually and exported as code.

## Product summary

This generator is designed for:

- designers building settings panels, preference screens, and control dashboards
- developers who want editable switch styling without rebuilding the component manually
- freelancers and agencies preparing branded binary controls for client UI
- buyers who want a standalone toggle tool consistent with the UI Foundry suite

## What the product includes

- section-based editing controls
- live preview panel
- code view with copy support
- file export support
- searchable preset library
- generated premium preset catalog with filters, paging, and surprise-me apply
- undo and redo controls
- self-contained standalone app structure
- native metadata and description controls for real switch authoring

## Editing coverage

The editor currently includes these control areas:

- Presets
- Basics
- Metadata
- Track
- Thumb
- States
- Description & Messages
- Effects
- Typography
- Accessibility

These sections cover switch setup, native metadata, track styling, thumb styling, state behavior, description and support messages, effects, label typography, and accessibility metadata.

## Export workflow

The toggle preview and code output reflect the same current switch state so the exported implementation follows the editor closely.

## Why this product is useful

This is more than a simple switch snippet. It gives the buyer a dedicated toggle studio for consistent binary control design.
